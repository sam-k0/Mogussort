# ЁЯУоHow it worksЁЯУо
There are 2 (two, 2ЁЯУо) tuples, I and O.
I contains all crewmates at the start of the game.
O is empty.

The algorithm will vote out one random crewmate and put it in O.
That crewmate is then removed from I.
Then, it checks if O is correctly sorted (ascending).

If it is correnct, it will continue to vote and populate O.

If the crewmates are not in the right order an ЁЯУоImposterЁЯУо(sus, baka) has been spotted.
This means that the game is lost, and O will be cleared, and I will be reset to the original form.

The algorithm will stop once all crewmates are in the right order.
